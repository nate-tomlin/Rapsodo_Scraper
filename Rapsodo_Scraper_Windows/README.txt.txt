Requirements:

-  Download and Install the Chrome web browers if it isn't already installed on the system (https://www.google.com/chrome/)
-  Download Chrome Driver (chromedriver_win32.zip) from: https://chromedriver.storage.googleapis.com/index.html?path=80.0.3987.16/
-  Extract from Zip Folder
-  Put chromedriver application into the C://Windows Directory
