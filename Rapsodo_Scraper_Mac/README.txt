Requirements:

-  Download and Install the Chrome web brower if it isn't installed on the system already (https://www.google.com/chrome/)
-  Download Chrome Driver (chromedriver_mac64.zip) from: https://chromedriver.storage.googleapis.com/index.html?path=80.0.3987.16/
-  Extract from Zip Folder
-  Put chromedriver executable into the /usr/local/bin Directory
